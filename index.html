視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓網站背景以比前景更慢的速度移動。 當訪客向下捲動時會產生 3D 視覺效果，營造出深度，並創造更沉浸的瀏覽體驗。 視差源於視覺錯覺：因為人眼會將近處的物體看得比遠處更大，我們也會覺得遠處的物體移動得更慢。 這種錯覺早已被運用在不同媒介上，讓畫面更貼近現實；最早可以追溯到迪士尼《白雪公主》等傳統動畫， 以及《超級瑪利歐》等早期遊戲。 隨著 CSS 與 HTML 的進步，視差效果逐漸走入現代的網頁設計世界。

最早在瀏覽器中實作視差滾動的案例之一，是 2007 年由開發者 Glutnix 在部落格分享的示範， 其中包含支援當時 Internet Explorer 6 等瀏覽器的 JavaScript 與 CSS2 範例碼與展示。 不過要到 2011 年，隨著 HTML5 與 CSS3 的普及，這項技巧才在網頁設計師之間流行起來。 支持者認為，這是擁抱網頁流動感的簡單方式，也能用來提升互動性與整體使用體驗。 但普渡大學 2013 年的研究也指出：雖然視差滾動強化了某些使用者體驗的面向， 卻不一定能提升整體的體驗。


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩個主要原因：

  • 有助於說故事

    視差圖像非常適合用來說故事。好的設計就是良好的溝通；在設計網站時，我們希望和訪客交流， 呈現產品、服務或品牌的精彩故事。視差滾動能讓訪客沉浸在獨特的線上體驗中， 甚至可以打造單頁網站，讓讀者不必切換頁面就能看完整個品牌故事。

  • 提升使用者參與度

    視差圖像也能提升使用者參與度。網站設計時需要注意降低跳出率──也就是使用者只瀏覽一頁就 返回或關閉分頁的比例。跳出率過高會影響搜尋排名，因此任何降低跳出率的作法都值得嘗試。 精心設計的視差頁面能帶來充滿動態感的體驗，讓使用者願意停留更久。


如果想在網站加入視差滾動效果，可以參考以下建議：

  • 適度使用

    視差滾動在重點元素上使用效果更好，不必放在所有頁面或所有地方。 可以將它應用在頁首、標題或首頁，讓視差成為強化設計的亮點，而不是分散使用者注意力。

  • 圖像壓縮

    視差滾動常使用多種媒體檔案與 CSS 圖形來營造深度。 過大的圖片會拖慢載入速度並影響體驗，因此記得壓縮影像、兼顧效能。

  • 運用色彩

    視差滾動不僅是前景與背景的圖像，色彩運用也同樣重要。 背景可以使用較為低彩度的色調，前景則採用鮮明色彩，能讓景深感更明顯。
